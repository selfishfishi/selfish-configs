Create a named session
`tmux new -s [session_name]` 

Attach to a session
`tmux attach -t [name]

Kill All
`tmux kill-server`

Create new window
`C-a c`

Switch to window
`C-a [window-no]`

View all windows
`C-a w`

Rename Window
`C-a ,`

Delete a window
`C-a &`

Last active window
`C-a l`

Previous/next window
`C-a p`
`C-a n`

Splits
`C-a s`
`C-a v`

Close current pane
`C-a x`

Last active pane
`C-a ;`

Switch Panes
`C-a {`
`C-a }`

Switch panes with number
`C-a q [number]`

Copy mode
`C-a [`
